# UI Package

Componentes Svelte reutilizables compartidos entre aplicaciones.
Biblioteca de componentes de interfaz de usuario comunes para mantener consistencia visual.

## Instalación

Este paquete se instala automáticamente como workspace dependency usando pnpm:

```bash
# Desde cualquier app
pnpm add @kidyland/ui --workspace
```

## Uso

```svelte
<script>
  import { Button, Card } from '@kidyland/ui';
</script>
```
