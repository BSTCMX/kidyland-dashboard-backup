# Admin Viewer App

Aplicación SvelteKit de solo lectura para visualización administrativa.
Permite consultar información del sistema sin capacidad de modificación.

## Creación

Esta app se creará usando pnpm:

```bash
cd apps/admin-viewer
pnpm create svelte@latest .
```

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm --filter admin-viewer dev

# O desde esta carpeta
pnpm dev
```
