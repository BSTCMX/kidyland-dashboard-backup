# Admin App

Aplicación SvelteKit para el Super Admin del sistema kidyland.
Permite gestión completa de usuarios, configuraciones, productos y todas las funcionalidades administrativas del sistema.

## Creación

Esta app se creará usando pnpm:

```bash
cd apps/admin
pnpm create svelte@latest .
```

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm --filter admin dev

# O desde esta carpeta
pnpm dev
```
