# Monitor App

Aplicación de visualización que muestra únicamente temporizadores.
Display dedicado para mostrar información de tiempo en pantallas públicas.

## Creación

Esta app se creará usando pnpm:

```bash
cd apps/monitor
pnpm create svelte@latest .
```

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm --filter monitor dev

# O desde esta carpeta
pnpm dev
```
