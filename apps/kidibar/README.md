# KidiBar App

Aplicación SvelteKit para la venta de productos en el KidiBar.
Interfaz de punto de venta para gestionar pedidos y transacciones del bar.

## Creación

Esta app se creará usando pnpm:

```bash
cd apps/kidibar
pnpm create svelte@latest .
```

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm --filter kidibar dev

# O desde esta carpeta
pnpm dev
```
