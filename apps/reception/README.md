# Reception App

Panel de recepción desarrollado en SvelteKit.
Interfaz para el personal de recepción para gestionar entradas, salidas y consultas básicas.

## Creación

Esta app se creará usando pnpm:

```bash
cd apps/reception
pnpm create svelte@latest .
```

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm --filter reception dev

# O desde esta carpeta
pnpm dev
```
